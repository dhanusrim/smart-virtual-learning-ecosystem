# Smart Virtual Learning Ecosystem

A MERN-based Learning Management System (LMS) that allows institutions to manage online courses, assessments, and student progress.

## Tech Stack

Frontend:
- React.js

Backend:
- Node.js
- Express.js

Database:
- MongoDB

Authentication:
- JWT
- bcrypt

---

## Features

• User authentication (Login/Register)  
• Role-based access (Admin, Faculty, Student)  
• Course management  
• Student enrollment  
• Assignment and quiz submission  
• Progress tracking  
• Dashboard visualization  
• Notifications and alerts  

---

## Installation

Clone the repository

git clone https://github.com/dhanusrim/smart-virtual-learning-ecosystem.git

Navigate to the project folder

cd smart-virtual-learning-ecosystem

Backend setup

cd backend  
npm install  
npm run dev  

Frontend setup

cd client  
npm install  
npm start
